# BOM-ABAP-RAP-project 
Overview
This is a custom SAP Fiori application built using the ABAP RESTful Application Programming Model (RAP). It provides a streamlined interface for business users to efficiently manage Bills of Material (BOMs) in bulk directly within SAP.

Key Features
Mass Excel Processing: Users can upload an Excel file to create or update multiple BOM lines simultaneously, eliminating manual data entry.

Full CRUD Capabilities: Standardized ability to view, create, edit, and delete BOMs directly from the Fiori Launchpad.

Modern Architecture: Powered by an OData V4 backend service and a Fiori Elements frontend for a seamless, responsive user experience.
